Graph module using Set-of-edges representation

Build a GraphSE.hs module that implements a Graph Abstract Data type using a Set-of-edges internal representation. Here's a type definition to get you started. This represents as a pair of a vertex set and and edge set. Assume that you will use this representation to implement both directed graphs (di-graphs) and un-directed graphs (graphs).

> newtype Graph a = Graph (Set a, Set (a,a))

Altogether your will build a Set module (the first part of this assignment), a Graph module using the Set module (this part of the assignment), and a top level GraphAlgsSimple.lhs module or script with some graph functions. This part of the lab is to construct just the Graph module.

If you want, you can start with the GraphSE-skel.lhs skeleton, a modified version of the Graph module skeleton given in class. (It is pretty rough at the moment.)

Be sure to test your GraphSE module using at least the tests given in the skeleton file.
